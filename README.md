# Simple HTML reader github
Ini adalah contoh sederhana cara membuat webserver pembaca html & css di github

- Buat repositori dengan format : {username}.github.io
- upload data html, css, img, dll ke repositori baru dengan format diatas. contoh seperti saya di https://github.com/aldiskatel/aldiskatel.github.io
- jika sudah tunggu beberapa saat github akan mengenali file kamu
- pastikan terdapat index.html untuk index file htmlnya
- jika sudah silahkan buka di {username}.github.io
